AI Resume Scrutiny Frontend
An AI-powered resume analysis platform that helps evaluate resumes through intelligent skill assessment, candidate ranking, and job-fit analysis. The application provides an interactive and responsive user interface for recruiters and hiring teams to review candidate profiles efficiently and gain actionable insights.

Resume analysis and scoring

Skill assessment and visualization
