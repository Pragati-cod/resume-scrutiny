# 📄 AI Resume Scrutiny System

An AI-powered resume screening and analysis platform that helps evaluate candidate resumes through intelligent skill assessment, resume scoring, and job-fit analysis. The system streamlines the recruitment process by providing actionable insights and candidate rankings through an interactive dashboard.

## 📌 Overview

This system simulates an intelligent recruitment assistant that:

* Uploads and analyzes resumes
* Extracts candidate information
* Evaluates skills and qualifications
* Generates resume scores
* Ranks candidates based on suitability
* Displays insights through a user-friendly dashboard

## ✨ Features

* Resume upload and parsing
* AI-powered resume analysis
* Skill and keyword extraction
* Candidate scoring and ranking
* Job-fit assessment
* Interactive and responsive dashboard
* Real-time insights and recommendations

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* GSAP
* AI/ML Integration

## 📂 Project Structure

```text
src/                 # Application source code

public/              # Public resources
```

## 🚀 Running the Project

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## 🎯 Objective

To simplify and accelerate the hiring process by leveraging AI-driven resume analysis, enabling recruiters to identify the most suitable candidates efficiently and make data-driven hiring decisions.
